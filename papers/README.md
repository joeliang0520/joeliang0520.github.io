Drop your paper PDFs here. They are linked from `assets/js/publications.js` via paths like `papers/2026-acl-scene-labeling.pdf`.

- `2026-acl-semantic-xpath.pdf` — Semantic XPath: Structured Agentic Memory Access for Conversational AI (ACL 2026 Demo)

- `2026-umap-vogue.pdf` — VOGUE: A Multimodal Dataset for Conversational Recommendation in Fashion (UMAP 2026)
- `2026-smr-mapping.pdf` — Mapping Social Change: A Unified Framework for Temporal Clustering (SMR)
